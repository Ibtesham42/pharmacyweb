# SEO Optimization Skill

## When to use
Auditing or improving any public page/post for search and Google Discover.

## Steps
1. Verify `generateMetadata`: title, description, canonical (absolute), OG, Twitter.
2. Verify JSON-LD type matches content (`JobPosting`/`Article`/`NewsArticle`/`BreadcrumbList`).
3. Check slug quality, breadcrumbs, and internal links to related content.
4. Ensure images use `next/image` + descriptive `alt`; LCP image prioritized.
5. Confirm inclusion in `sitemap.ts`; correct robots/index directives.
6. Validate structured data (Rich Results Test) and meta length limits.

## Checklist
- [ ] Metadata + canonical + OG/Twitter complete.
- [ ] Valid JSON-LD for the content type.
- [ ] In sitemap; index/noindex correct.
- [ ] Internal links + alt text present.
